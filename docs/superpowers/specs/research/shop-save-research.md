# Phase 6 调研：商店/存档/任务/成就品类分析

## 调研对象

| 游戏 | 持久化机制 |
|------|-----------|
| **Vampire Survivors** | 金币在局间保留，购买武器/角色的永久解锁，成就系统 |
| **Brotato** | 金币购买永久属性加成，角色解锁，波次间商店 |
| **H5 项目（基准）** | 灵魂碎片(30%掉落)购买6种永久升级(各3级)，14任务+27成就，跨局存档 |

## 值得引入

1. **跨局存档**（品类通用）：用 Godot ConfigFile 存储到用户目录
2. **商店永久升级**（H5 已有）：6 种升级 × 3 级，用灵魂碎片购买
3. **任务系统**（H5 已有）：14 个条件型任务，完成后奖励灵魂碎片
4. **成就系统**（H5 已有）：27 个里程碑/挑战成就，增加长期目标

## 调研结论

H5 的商店/任务/成就系统已经非常完善。直接复用数值，用 Godot ConfigFile 实现存档。灵魂碎片 = 金币 × 30% 转化率。
