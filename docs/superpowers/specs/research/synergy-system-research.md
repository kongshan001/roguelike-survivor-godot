# Phase 5 调研：协同效应系统品类分析

## 调研对象

| 游戏 | 协同机制 |
|------|---------|
| **Vampire Survivors** | 武器+被动道具进化（被动不直接产生协同效果，仅作为进化条件） |
| **Brotato** | 武器间无协同，靠装备属性叠加形成 build |
| **Holocure** | 角色专属协同，特定武器组合产生额外效果 |
| **H5 项目（基准）** | 17 种协同：6 被动+被动、12 武器+被动，实时生效 |

## 值得引入的机制

1. **被动+被动协同**（H5 已有）：6 种双被动组合产生独特效果
2. **武器+被动协同**（H5 已有）：12 种武器强化效果，鼓励特定搭配
3. **实时检测**（品类趋势）：每次获得/失去被动时重新计算活跃协同
4. **视觉提示**（体验优化）：协同激活时 HUD 显示图标

## 调研结论

H5 的 17 种协同系统设计成熟，直接复用。协同效果分为三类：属性加成、触发效果、行为改变。实时检测机制保证协同在获得/失去时即时生效。
