# Phase 3 调研：敌人系统品类分析

## 调研对象

| 游戏 | 敌人设计特点 |
|------|-------------|
| **Vampire Survivors** | 敌人行为单一（纯追踪），通过数量和组合制造压力。Boss 有特殊形态和AoE |
| **Brotato** | 波次制，每波敌人类型固定，玩家在波次间购买装备。敌人有冲锋/自爆/远程等行为 |
| **Holocure** | 远程敌人较多，弹幕密度高，Boss 有多阶段和复杂攻击模式 |
| **Magic Survival** | 敌人类型最少但数值曲线陡峭，后期纯靠数值压制 |
| **H5 项目（基准）** | 7 种敌人（僵尸/蝙蝠/骷髅/精英骷髅/幽灵/分裂者/小分裂者），Boss 单阶段，5 段波次进度 |

## 值得引入的机制

1. **远程敌人弹幕**（H5 已有）：骷髅单发/精英 3 方向，增加空间压力
2. **幽灵相位转移**（H5 已有）：短暂消失后瞬移到玩家附近，制造突发威胁
3. **分裂者**（H5 已有）：死亡分裂为 2 个小分裂者，增加后期清怪压力
4. **Boss 多阶段**（品类趋势）：VS/Holocure 都使用多阶段 Boss，提高战斗高潮感
5. **无尽模式递增**（H5 已有）：Boss 每 240s 刷新，HP/速度递增，提供长期挑战

## 不引入的机制

- 敌人 AI 寻路/绕障（项目无障碍物，纯追踪足够）
- 敌人间的协作行为（复杂度过高，收益低）
- 自爆敌人（与分裂者功能重叠）

## 调研结论

H5 的 7 种敌人设计已经覆盖了品类核心玩法。额外引入 Boss 三阶段行为（追击→冲锋+弹幕→极速+螺旋弹幕）提升战斗高潮感。波次进度和无尽模式递增机制保持与 H5 一致。
