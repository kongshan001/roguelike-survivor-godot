## 1. Executive Summary

v1.0.2 是 v1.0.1 hotfix 之后的首个内容与打磨更新。核心目标为：改善中期升级节奏（XP 曲线微调）、延长 meta-progression 养成周期（商店 T4 + 武器精通）、增强战斗 gamefeel（击中反馈 + 投射物拖尾）、以及视觉打磨（角色动画 + 敌人动画）。项目评分从 94.6 提升至 96.4 (+1.8)，测试从 1276 增长至 1635 (+359, +28%)。

## 2. New Features

### 2.1 XP Curve Tuning (R19 design, R20 implementation)

**Design Spec**: `docs/superpowers/specs/xp-curve-tuning.md`

**Change**: EXP_TABLE 索引 4-6 (Lv5->8) XP requirements reduced ~10%.
- Index 4: 32 -> 29
- Index 5: 42 -> 38
- Index 6: 55 -> 50

**Impact**: Wave 2 upgrade count increased from 2 to 3 (+50%). Lv8 arrival time reduced from 2:30 to 2:11 (-12.7%). "Flat middle" compressed from ~60s to ~50s.

**Files changed**: `scripts/autoload/game_manager.gd` (1 line)
**Tests added**: 31 (test_xp_curve_tuning.gd)

### 2.2 Shop Tier 4 Upgrades (R19 design, R20 implementation)

**Design Spec**: `docs/superpowers/specs/shop-t4-design.md`

**Change**: All 6 shop upgrades extended to 4 tiers. T4 cost = 2x T3 cost.

| Upgrade | T4 Cost | T4 Effect | Total Cost (T1-T4) |
|---|---|---|---|
| maxhp | 160 | +5 HP | 300 |
| speed | 160 | +5% | 300 |
| pickup | 120 | +5 range | 225 |
| expbonus | 200 | +5% | 375 |
| weapondmg | 240 | +5% | 450 |
| gold | 120 | +10% | 225 |

**Total shop cost**: 875 -> 1875 (+114%). Full upgrade runs: ~5 -> ~10.8.

**Files changed**: `scripts/autoload/save_manager.gd` (SHOP_UPGRADES, 6 getter functions), `scripts/shop.gd` (effect text)
**Tests added**: 39 (test_shop_t4.gd)

### 2.3 Weapon Mastery System (R19 design, R20 implementation)

**Design Spec**: `docs/superpowers/specs/weapon-mastery.md`

**Change**: Kill-tracking progression for 7 base weapons, 4 mastery tiers per weapon.

| Tier | Name | Kill Threshold | Damage Bonus |
|---|---|---|---|
| 0 | Novice | 0 | +0% |
| 1 | Apprentice | 50 | +2% |
| 2 | Adept | 200 | +4% |
| 3 | Expert | 500 | +6% |
| 4 | Master | 1000 | +8% |

**Key mechanics**:
- Evolved weapons credit both parent weapons simultaneously
- Bonus is additive with shop weapondmg, multiplicative with character passive
- Maximum combined bonus (T4 shop + Tier 4 mastery + Mage): 1.48x (healthy)
- 2 new achievements: mastery_first (30 SF), mastery_all (500 SF)

**Files changed**: `scripts/autoload/save_manager.gd` (mastery constants/functions/save/load), `scripts/enemy.gd` (kill attribution with evolved_parents map), `scripts/weapon_controller.gd` (mastery damage multiplier)
**Tests added**: 52 (test_weapon_mastery.gd)

### 2.4 Character Animation (R18 implementation)

**Design Spec**: `docs/superpowers/specs/character-animation-integration.md`

**Change**: 2-frame walk cycle at 4 FPS using Method C (Sprite2D + _physics_process). No .tscn modifications.

**Files changed**: `scripts/player.gd` (+62 lines: ANIM_INTERVAL, _anim_time, _anim_frame, _idle_texture, _action_texture, _setup_character_animation, _load_texture_safe)
**Tests**: Existing 1319 tests verified, 3 initially pending (resolved)

### 2.5 Enemy Animation System (R19 implementation)

**Design Spec**: `docs/superpowers/specs/enemy-animation-spec.md`

**Change**: 10 differentiated death effects per enemy type. Hit feedback using Tween-driven HDR white flash + position jitter. Elite enemies get residual color tint after hit.

**New file**: `scripts/enemies/enemy_death_effects.gd` (249 lines)
**Files changed**: `scripts/enemy.gd` (refactored from manual flash_timer to Tween-driven feedback)
**Tests added**: 79 (R19 total)

### 2.6 HUD Upgrade Card Hover Effect (R19 implementation)

**Change**: Scale + modulate Tween on card mouse_entered/exited. HBoxContainer layout limitation means no Y-offset (documented as accepted).

**Files changed**: `scripts/hud.gd` (374 -> 410 lines)

### 2.7 Hit Feedback System (R20 design, R21 implementation)

**Design Spec**: `docs/superpowers/specs/hit-feedback-design.md`

**New file**: `scripts/effects/projectile_trail_pool.gd` (125 lines, Node)

## 7. Upgrade Guide (v1.0.1 -> v1.0.2)

### Save File Compatibility

v1.0.2 save files are **forward compatible** with v1.0.1 saves:

- **Shop T4**: Old saves with shop levels 0-3 load correctly. max_level extended from 3 to 4. T4 purchases are new.
- **Weapon Mastery**: Old saves without `[mastery]` section default to 0 kills / Tier 0 for all weapons. No migration needed.
- **Achievements**: 2 new achievements (mastery_first, mastery_all) added to ACHIEVEMENTS array. Existing achievement state preserved.
- **XP Curve**: EXP_TABLE change only affects live gameplay, not saved data. No migration.

### API Changes

| Change | File | Impact |
|---|---|---|
| `SaveManager.get_*_bonus()` return arrays extended to 5 elements | save_manager.gd | Breaking if callers index by hardcoded position |
| `SaveManager.add_weapon_kill(weapon_id)` new method | save_manager.gd | Additive, no breaking change |
| `SaveManager.get_weapon_mastery_tier/bonus(weapon_id)` new methods | save_manager.gd | New API |
| `enemy.gd die()` refactored to delegate to enemy_loot module | enemy.gd, enemy_loot.gd | Internal restructure, external interface unchanged |
| `hit_feedback` module attached to enemy take_damage | enemy.gd | Additive visual layer |

### Configuration Constants Added

| Constant | File | Value |
|---|---|---|
| MASTERY_THRESHOLDS | save_manager.gd | [0, 50, 200, 500, 1000] |
| MASTERY_BONUSES | save_manager.gd | [0.0, 0.02, 0.04, 0.06, 0.08] |
| BASE_WEAPONS | save_manager.gd | ["knife","holywater","lightning","bible","firestaff","frostaura","boomerang"] |
