# Phase 6 脑洞：商店/存档创新方案

## 方案 1：H5 原版商店 + ConfigFile 存档（推荐 ★★★）
- **核心创意**：6 种永久升级用灵魂碎片购买，Godot ConfigFile 存储，局间保留
- **玩家体验**：每局都有成长感，即使失败也能变强
- **可行性**：技术成本低，Godot ConfigFile 原生支持

## 方案 2：云存档（不推荐 ★）
- **问题**：需要服务器，单机项目不需要

## 方案 3：Roguelike 遗产系统（备选 ★★）
- **核心创意**：死亡时保留部分武器等级到下一局
- **问题**：与 H5 设计差异大，增加平衡复杂度

## 收敛

采用方案 1。ConfigFile 存储全部持久化数据。
