# Phase 4 脑洞：武器进化创新方案

## 方案 1：双武器合成进化 + 进化特效（推荐 ★★★）
- **核心创意**：两个 Lv3 武器合成 → 一个进化武器，进化时全屏闪光+武器消失动画
- **融合点**：复用现有 WeaponData 类型系统，进化武器使用组合类型
- **玩家体验**："质变"感强，鼓励尝试不同武器组合
- **可行性**：技术成本低（扩展 upgrade_pool + weapon_controller），数值平衡已由 H5 验证

## 方案 2：进化武器可继续升级（不推荐 ★）
- **核心创意**：进化后武器仍可升级到 Lv3，获得更强效果
- **问题**：与 H5 设计矛盾，后期数值膨胀难控制，增加大量设计工作量

## 方案 3：条件触发进化（备选 ★★）
- **核心创意**：除双武器外，增加特殊条件进化（如击杀 Boss 解锁隐藏进化）
- **融合点**：可作为后续迭代内容
- **问题**：当前阶段复杂度过高

## 方案收敛

**采用方案 1**，直接复用 H5 的 8 种进化配方。进化特效通过简单的画面闪烁实现。
