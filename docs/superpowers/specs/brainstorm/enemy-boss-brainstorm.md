# Phase 3 脑洞：敌人系统创新方案

## 方案 1：Boss 三阶段行为（推荐 ★★★）
- **核心创意**：Boss 战分为三个阶段，每阶段 HP 降到阈值后切换行为
  - Phase 1 (100%-66%)：纯追踪玩家，速度适中
  - Phase 2 (66%-33%)：周期性冲锋（速度×3，持续0.8s），同时向 8 方向发射弹幕
  - Phase 3 (33%-0%)：速度提升50%，同时向 16 方向发射螺旋弹幕
- **融合点**：复用已有 enemy_bullet.gd 弹幕系统，phase 切换通过 HP 阈值触发
- **玩家体验**：战斗节奏从"躲避大怪"升级到"弹幕地狱"，成就感强
- **可行性**：技术成本低（扩展现有 enemy.gd），数值平衡中等（需调 boss HP/伤害）

## 方案 2：元素敌人（不推荐 ★）
- **核心创意**：敌人带元素属性（火/冰/雷），与玩家武器产生交互
- **问题**：与 Phase 5 协同系统功能重叠，增加过多复杂度

## 方案 3：精英敌人变体（备选 ★★）
- **核心创意**：每种基础敌人有低概率出现精英变体（更大、更强、掉落更多）
- **融合点**：复用已有 EnemyData，通过精英标记调整数值
- **问题**：当前 Phase 3 范围已经较大，可作为 Phase 3.5 后续迭代

## 方案收敛

**采用方案 1**（Boss 三阶段行为），方案 3 留作后续迭代。

Boss 三阶段提升战斗高潮感，技术实现可控，与 H5 的单阶段 Boss 形成差异化提升。
