# Phase 5 脑洞：协同效应创新方案

## 方案 1：H5 原版 17 协同 + HUD 提示（推荐 ★★★）
- **核心创意**：复用 H5 的 17 种协同配方，增加 HUD 图标显示激活状态
- **融合点**：复用已有 owned_passives 和 owned_weapons 字典做实时检测
- **玩家体验**：每次获得被动时可能触发协同，增加"发现组合"的乐趣
- **可行性**：技术成本低（新建 synergy_manager.gd），数值已由 H5 验证

## 方案 2：三件套协同（不推荐 ★）
- **核心创意**：3 个被动同时持有时触发更强大的协同
- **问题**：触发概率太低（需要凑齐 3 个特定被动），玩家体验差

## 方案 3：协同升级（备选 ★★）
- **核心创意**：协同效果本身可以升级（叠加更多被动强化协同）
- **问题**：增加过多复杂度，留作后续迭代

## 方案收敛

**采用方案 1**。17 种协同直接复用 H5 数据，HUD 显示留作 UI 增强。
