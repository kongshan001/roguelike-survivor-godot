## 更新规则

- 每个 Round 结束后，PM Agent 追加记录到当前开发版本的详细文件
- 版本发布时标记状态为"已发布"，添加发布日期
- 每次修改后 `git add + commit + push` 到远程仓库
- 格式：Round 级别，中文为主，技术术语保留英文
